# server

- server의 루트 디렉터리인 agent-server로 이동해서 실행해야 함
- 가상환경 실행 (예: source .venv/bin/activate)
    - 이때 가상환경(예: .venv)은 server 루트 디렉터리(TETS/agent-server)에 위치해야 함
- 의존성 설치
    - `pip install -e requirements.txt`
    - 오류 발생 시 ``
- 실행 명령 : `uvicorn src.main:server --host 0.0.0.0 --port 8123 --reload`
- 서버 실행 : 아래 로그가 뜨면 서버 실행을 성공한 것
    ```
    INFO:     Started server process [89875]
    INFO:     Waiting for application startup.
    INFO:     Application startup complete.
    ```
